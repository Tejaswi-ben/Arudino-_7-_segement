# Arudino-_7-_segement
Arduino 7-Segment Display Simulation

This project demonstrates how to interface a Common Cathode 7-Segment Display with an Arduino Uno using digital output pins. The simulation is developed and tested in Wokwi, allowing the circuit to run without physical hardware.

The Arduino controls each segment (A–G) individually to display decimal digits from 0 to 9. The project helps beginners understand GPIO pin control, digital output, and the working principle of a 7-segment display. It is a simple embedded systems project suitable for students learning Arduino programming and electronics.
